IE8부터 모바일웹까지 고려하여 jQuery UI 사용하기

created
2019년 7월 4일
category
worklog
tags
IEJavascriptjQuerymobile

드래그앤 드랍을 써야하는데 하나의 마크업으로 IE8부터 모바일웹까지 모두 고려해야하는 요구사항이 있어서 선택의 여지 없이 jQuery UI를 사용하게 됐다.

jQuery도 IE 하위버전까지 다 지원하려면 1점대 버전을 사용해야 하듯이 jQuery UI도 버전이 오르다보니 IE8부터 모바일까지 다 맞추려면 낮은 버전을 선택해야 했고 jQuery UI만으로 모바일까지 당연히 대응이 될 것이라고 예상했는데 예상 외로 모바일까지 적용하려면 추가 플러그인이 필요했다.

필요한 jQuery 라이브러리와 플러그인 버전

  • jQuery 1.7.2 이하
  • jQuery UI 1.8.21 이하
  • jQuery UI Touch Punch : 모바일에도 동일하게 적용하기 위한 플러그인 (사용버전은 0.2.3 이지만 상관없을듯)
