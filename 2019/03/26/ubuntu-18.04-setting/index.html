18.04 설치 후 할일

설치 후 바로 업데이트

$ sudo apt update
$ sudo apt upgrade

알고 설치

  • Chrome
  • VSCode
  • Sublime Text3 - https://www.sublimetext.com/docs/3/linux_repositories.html
  • Sublime Merge - https://www.sublimemerge.com/docs/linux_repositories
  • Node.js - https://github.com/nodesource/distributions/blob/master/README.md
  • git - sudo apt install git
  • npm - sudo apt install npm
  • slack
    제공 사이트가 있거나 필요한건 그냥 커맨트 쳐보면 apt install 을 입력하라고 나오니 편함..

잘 모르는데 설치

  • tlp sudo apt install tlp
  • gdebi
  • gufw
  • gnome usage

보류

  • 미디어 코덱
    • sudo apt install ubuntu-restricted-extras
  • gnome shell extension
    • https://extensions.gnome.org/
    • https://chrome.google.com/webstore/detail/gnome-shell-integration/gphhapmejobijbbhgpjhcjognlahblep

추가세팅

  • 한/영 전환 - 우측 위 아래 화살표 누른 후 한글(Hangul)로 변경
  • 한글폴더 영문으로 변경
    • ~/.config/user-dirs.dirs - 폴더를 미리 만들어두지 않으면 재부팅시 다 $Home으로 잡힘
  • CapsLock - Super 키로 설정하기
    • gnome tweak sudo apt-get install gnome-tweak-tool
    • 기능개선 - 키보드와 마우스 - 추가 배치 옵션 - Caps Lock 동작
