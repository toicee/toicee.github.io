맥 터미널 세팅 iTerm2 & zsh

created
2017년 11월 20일
updated
2018년 10월 15일
category
etc
tags
MacTerminal

9월말에 받은 iMac을 세팅할 새가 없어 켜보기만 하다가 새 프로젝트 들어가기전 터미널부터 세팅했다.
세팅은 어쩌다가 하니까 매번 검색 해야되는데 이후엔 그냥 이거 참고하려고 정리

iTerm2 설치

brew install cask iterm2

ZSH 설치

brew install zsh

oh-my-zsh 플러그인 설치

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

컬러셋

https://iterm2colorschemes.com/

oh-my-zsh 컬러셋

https://github.com/robbyrussell/oh-my-zsh/wiki/Themes

플러그인

plugins=(
  git
  sublime
  zsh-syntax-highlighting
  zsh-autosuggestions
)

git clone https://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions

플러그인 설치는 위 같은 방법으로 플러그인 경로에 바로 내려 받는게 편리하다.

단축키 설정

alias 단축키="실행문"
alias hosts="sudo nano /private/etc/hosts"
